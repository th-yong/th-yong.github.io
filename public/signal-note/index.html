정말 무료인가요?

네. 개인적으로 만든 프로젝트라 별도 과금은 없습니다. 인프라·LLM 비용은 운영자가 부담합니다.

신청했는데 확인 메일이 안 와요.

Gmail 사용자라면 스팸함 / 프로모션 탭을 먼저 확인해 주세요. (발신 도메인이 Azure Communication Services라 처음엔 스팸으로 분류될 수 있어요.) 해당 메일에 "스팸 아님"을 눌러주시면 다음부터는 받은편지함으로 도착합니다. 그래도 안 오면 yongtaehoon@gmail.com 으로 연락 주세요.

해지는 어떻게 하나요?

메일 푸터의 "unsubscribe" 링크 또는 해지 Google 폼으로 이메일을 제출하시면, 확인 메일의 버튼 클릭과 함께 즉시 목록에서 제거됩니다.

키워드를 개인별로 바꿀 수 있나요?

현재는 모든 구독자에게 동일한 키워드 세트(AI/ML 엔지니어 기본 49개 동의어 그룹)로 큐레이션합니다. 개인화는 향후 로드맵에 있어요.

내 이메일은 어떻게 보관되나요?

Private GitHub Gist에 주소만 평문으로 저장되며, 접근 가능한 사람은 운영자 1인입니다. 발송 시에는 수신자별 개별 메일로 보내기 때문에 서로의 주소는 노출되지 않습니다.

왜 메일인가요? Slack/Discord 안 되나요?

이메일은 포맷·보관·검색이 가장 자유로워서 선택했습니다. 수요가 있으면 다른 채널도 고민해볼게요.
